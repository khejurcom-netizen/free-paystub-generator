## Responsible Use

Paystub generators should be used only to create accurate payroll records, payment statements, contractor documentation, and personal bookkeeping records.

* Enter only truthful employer, employee, earnings, tax, and deduction information.
* Check federal, state, and local payroll rules before issuing a paystub.
* Use a registered payroll provider or qualified payroll professional where certified payroll records, tax filing, or statutory reporting is required.
* Do not create, edit, or submit a paystub to misrepresent income, employment, or tax information.
